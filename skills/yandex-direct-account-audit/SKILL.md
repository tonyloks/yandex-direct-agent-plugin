# yandex-direct-account-audit (v1 skeleton)

Use this skill to produce a concise account audit.

1. Run an audit of the Yandex Direct account.
2. Check campaign structure and consistency.
3. Review ads, keywords, negative keywords, ad extensions, and landing pages.
4. Perform a separate 404 check for landing pages.
5. Perform a separate feed health check.
6. If required data is missing, fetch it first via MCP tools.
7. Return a structured audit summary with findings, severity, and next actions.
