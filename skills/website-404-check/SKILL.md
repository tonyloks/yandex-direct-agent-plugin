# website-404-check (v1 skeleton)

Use this skill to perform a fast page availability check.

1. Verify that the page opens.
2. Verify the final HTTP status code.
3. Verify redirect chain behavior.
4. Verify whether the page looks like a soft-404.
5. Record the result in a short report.
